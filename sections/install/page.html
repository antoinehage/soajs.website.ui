Installation Steps


Follow the steps below to download the SOAJS Installer and use it to deploy the SOAJS Console.


1

Download and Unzip

  1. Download the SOAJS installer Zip file
  2. Unzip/Extract the download on your machine
2

Install

  1. Open your terminal and navigate to the extracted folder
  2. Run the installation script
    > cd soajs.installer
> ./INSTALL
3

Deploy the Console

  • Use the installer and deploy the SOAJS Console via the following command:
    > soajs console install %folder_path%
  • Once deployed, the installer displays on your terminal a URL to open in your browser.
    $ In your Browser, open: http://localhost:4005/
4

Documentation & Manual

  • The installer is equipped with a manual: 
    > soajs --help
  • Check out the full documentation available on SOAJS Confluence Space.
